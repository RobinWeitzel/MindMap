Mind Mapping Tool

  • Double click to create and edit elements
  • Drag with left mouse to create connections
  • Delete objects via the delete-key
  • Change text size via "alt +" and "alt -"
  • Change color via "alt <starting letter of color>" (i.e. "alt o")
  • Create a new mind map with "alt n"
  • Open an existing mind map with "alt o"
